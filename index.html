Selamat Pagi CNC

Selamat Pagi CNC merupakan pelatihan gratis yang diadakan setiap tahun oleh Computer and Communication Laboratory.
Materi pelatihan ini mengenai web microblogging menggunakan fitur geolocation Google MAPS API, backend menggunakan CodeIgniter (Credit to http://www.codeigniter.com) serta frontend menggunakan framework CSS MaterializeCSS (Credit to http://www.materializecss.com)

Petunjuk Instalasi

  • Download file mysql pada link di bawah ini: https://db.tt/XjSXCetW
  • Lakukan konfigruasi database pada config/database. Sesuaikan nama databasenya.
  • Lakukan import database pada phpmyadmin
  • Pastikan base_url sama dengan nama folder project Anda yang berada pada htdocs.

Note: Jika membutuhkan modul tersedia pada link berikut: https://db.tt/XbZKPyol
